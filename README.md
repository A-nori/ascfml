# ascfml
You can write propositional formula
in ascii-art.

## usage
`\ascfml{formula}`

### example
`\ascfml{=> ~~A -> A}  
\ascfml{=> A \/ ~A}  
\ascfml{A /\ (B \/ C) => (A \/ B) /\ (A \/ C)}  

You can use =, /, \ and - for usual usage.  
\ascfml{A = B /\ B = C => A = C}  
\ascfml{A/B - C => \varphi}`
